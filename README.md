# Coffee-Sales-Excel-Project
Analyzed 1,000+ coffee orders (2019–2022) across the US, Ireland &amp; UK using Excel. Used XLOOKUP &amp; INDEX/MATCH for data enrichment, then built PivotTables to uncover sales trends by coffee type, top customers &amp; regional performance  visualized in an interactive dashboard with slicers and timeline filters.
Insights:
Data enrichment using XLOOKUP & INDEX/MATCH across orders, customers & products tables
Monthly sales breakdown by coffee type (Arabica, Excelsa, Liberica, Robusta)
Top 5 customers by spend & country-wise revenue comparison
Interactive dashboard with PivotTables, slicers & timeline filter
